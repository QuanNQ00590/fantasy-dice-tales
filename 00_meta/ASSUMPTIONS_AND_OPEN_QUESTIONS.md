# Assumptions And Open Questions

## Current Assumptions

- Game is mobile-first portrait orientation unless later changed.
- Primary input is tap to roll and tap to resolve tile events.
- Dice rolls consume an energy-like resource or dice token.
- Board map is the main session surface.
- Fantasy theme replaces ocean/Atlantis theme.
- Art target is cartoon-border: bold outlines, readable silhouettes, bright fantasy colors.
- Prototype should focus on one map region and a limited tile set.
