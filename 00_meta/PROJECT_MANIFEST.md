# Project Manifest

## Identity

- Working title: Fantasy Dice Tales
- Genre: Casual board-game adventure with dice movement
- Theme: Fantasy exploration
- Visual target: Cartoon-border, readable, colorful, characterful
- Platform target: Mobile first
- Business target: Free-to-play with optional IAP/RV/liveops

## New Direction

This project intentionally moves away from the previous slot-machine centered Atlantis Fortune direction.

Old center:

`Energy -> Slot -> Reward -> Attack/Steal/Fishing -> Build`

New center:

`Energy/Dice -> Roll -> Move on board -> Trigger tile -> Resolve event -> Reward -> Unlock map/story/progression`

## Design Pillars

1. Dice-driven anticipation
   - Every roll should create suspense before movement resolves.

2. Map discovery
   - The board is not just a reward surface. It is the main play space.

3. Light tactical choice
   - Players should have small decisions: use dice item, choose route, stop at event, spend resource, risk combat.

4. Fantasy charm
   - Characters, landmarks, tiles, creatures, and rewards should feel like a playful fantasy storybook.

5. Production-safe scope
   - Prototype must prove the board loop before adding complex social, deep RPG, or heavy liveops.

## Non-goals For First Prototype

- No full open-world simulation.
- No complex tactical combat grid.
- No heavy guild/social war system.
- No deep RPG equipment system.
- No production monetization tuning before core loop validation.

## Source Safety

Original Atlantis Fortune files and GDD Template files must remain unchanged. New documents and data live only inside `G:\Fantasy Dice Tales`.
