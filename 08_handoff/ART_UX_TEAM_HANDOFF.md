# Art And UX Team Handoff

## Art And UX Mission

Create a readable, charming fantasy board presentation with strong tile clarity and fast session comprehension.

## Art And UX Priorities

- clear tile readability at mobile size
- readable movement anticipation before landing
- strong differentiation between reward, risk, gate, and utility tiles
- fantasy charm without visual noise
- tutorial readability in first-session flow

## Expected Output From Art And UX

- tile visual language proposal
- board readability rules
- first-region mood and landmark direction
- HUD and roll button clarity pass
- animation and feedback priorities for prototype

## Minimum Inputs Required Before Art Production Starts

- first region theme is locked
- tile taxonomy is locked for the milestone
- required HUD elements are locked
- board camera scale and target mobile aspect are agreed
- hero asset list for the prototype is approved
