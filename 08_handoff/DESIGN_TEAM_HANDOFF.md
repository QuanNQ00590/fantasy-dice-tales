## Design Deliverables Expected Next

- resolved board topology
- first-region node flow
- initial tile set and event outcomes
- reward pacing for first-session loop
- branch choice rules, if any

## Decisions Blocking Prototype Quality

- how much agency players have over dice outcomes
- whether branching paths exist in v1 prototype
- how combat resolves inside the board loop
- what the first long-term progression sink is
