# Reference Notes

## Reference Direction

The project should study board-game dice adventure pacing similar to Mini Tales-like experiences:

- Dice roll anticipation
- Node/path map movement
- Landing tile events
- Short-session reward loops
- Board completion and map unlocking

This document is intentionally descriptive rather than a source clone. Do not copy layouts, UI, IP, characters, exact tile distributions, or monetization details from any single reference.

## Art Reference Keywords

- Fantasy cartoon-border
- Storybook board
- Bold outline
- Cozy magic
- Chunky readable props
- Cute character token
- Colorful map path
- Soft painterly background

## Gameplay Reference Keywords

- Dice board
- Map exploration
- Tile event
- Branch choice
- Chest milestone
- Region unlock
- Character helper
- Event board

## Things To Avoid

- Slot-machine identity
- Overly punitive random losses
- Hidden dice rigging
- Board that feels like a plain progress bar
- Text-heavy story panels
- Too many currencies in prototype
