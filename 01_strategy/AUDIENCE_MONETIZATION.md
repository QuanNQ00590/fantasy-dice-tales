# Audience And Monetization

## Target Audience

Primary:

- Casual mobile players who enjoy board progression, reward anticipation, collections, and light fantasy.

Secondary:

- Players from casual dice/board/event games who want a softer fantasy adventure tone.

Audience needs:

- Fast clarity
- Frequent rewards
- Low punishment
- Cute/fantasy art
- Visible progress
- Optional optimization for engaged players

## Session Design

Micro session:

- 1-3 rolls
- Claim a reward
- Spend on one upgrade or map unlock

Normal session:

- 10-20 rolls
- Complete a short board segment
- Trigger one special tile or encounter

Long session:

- Use saved dice/energy
- Complete event milestones
- Optimize route or rewards

## Monetization Principles

- Monetization should increase agency, pace, or collection joy.
- Dice purchases must not feel like buying random loss.
- RV should be optional and clearly valuable.
- Event pass should add structured goals, not replace base progression.

## Potential Offers

- Dice pack
- Energy refill
- Special dice item bundle
- Event board pass
- Map starter pack
- Character cosmetic bundle
- Reward doubler via RV

## Risks

- If dice supply is too tight, the game feels blocked.
- If paid dice dominate, free users will churn early.
- If rewards are too random, players will not trust progression.
- If map completion requires too many rolls, the board becomes a grind surface.
