# Roadmap

## V0 Discovery

Goal:

- Confirm direction, references, art target, and board topology.

Deliverables:

- Project brief
- GDD master
- Board map system
- Dice system
- Art direction
- Prototype scope

## V1 Prototype

Goal:

- Prove roll -> move -> tile -> reward -> progress.

Deliverables:

- One playable board
- Normal dice
- Tile event system
- Reward delivery
- Basic upgrade
- First-session tutorial
- P0 tracking

## V2 Vertical Slice

Goal:

- Make the game feel like a small finished product slice.

Deliverables:

- Polished Greenvale board
- 2-3 board segments
- 1 event board draft
- Character token
- Chest and gate flow
- Economy tuning pass
- Art/audio pass

## V3 Pre-soft-launch

Goal:

- Validate retention and economy.

Deliverables:

- 3+ regions or boards
- Liveops event loop
- Shop/RV/IAP stub
- Full tracking dashboard
- Balance simulation
- QA checklist

## V4 Soft Launch

Goal:

- Measure D1/D3/D7, economy, monetization, content pacing.

Deliverables:

- Server config
- Event calendar
- Telemetry dashboards
- Economy control alerts
- Content update workflow
